## Intro

Hi Developers 

This is a **Recipe Book Desktop Application** built using **Python and Tkinter**, designed to store, manage, and view your favorite recipes. With a user-friendly interface and a local SQLite database, it's the perfect tool for organizing your cooking creations.


##  Features

-  Add New Recipes  
-  View Full Recipe Details  
-  Update Existing Recipes  
-  Delete Recipes You No Longer Need  
-  Local Data Storage with SQLite  
-  Attractive and colorful GUI


##  Requirements

- Python 3.x  
- Tkinter (comes built-in with Python)  
- SQLite (also built-in)

> No external packages needed!
